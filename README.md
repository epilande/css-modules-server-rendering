# YARIS
**Yet Another React ~~Isomorphic~~ Universal Starter**

If you don't want Universal JavaScript, check out my other boilerplate **[YARS](https://github.com/epilande/yars)**.

## Features
- [X] Server-Side Rendering
- [X] Webpack v2
- [X] React
- [X] React HMR
- [X] React Redux
- [X] React Router (async routes)
- [X] React storybook
- [X] Babel
- [X] Styled-components
- [X] Jest and Enzyme
- [X] ESLint
- [X] Stylelint
- [X] NPM Scripts

## Quick Start

#### Clone this repo

```bash
$ git clone https://github.com/epilande/yaris.git
$ cd yaris
```

#### Install dependencies

```bash
$ npm install
```

#### Launch dev environment

```bash
$ npm run dev
```

#### Build prod

```bash
$ npm run build
```

#### Start prod server

```bash
$ npm run start
```
