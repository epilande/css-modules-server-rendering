## Quick Start

#### Clone this repo

```bash
$ git clone https://github.com/epilande/yaris.git
$ cd yaris
```

#### Install dependencies

```bash
$ npm install
```

#### Launch dev environment

```bash
$ npm run dev
```

#### Build prod

```bash
$ npm run build
```

#### Start prod server

```bash
$ npm run start
```
